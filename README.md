# heading
for study
使用maven建立了父模块-子模块结构的一个工程，并在不同的子模块中进行技术学习

1.heading-core：核心模块，涵括了一些common代码；
2.heading-spring：spring模块，用于学习spring技术
2.1 spring整体搭建和简单访问，访问类HelloworldController,该类实现了切面编程，关联类TransactionDemo；
2.2 自定义注解的实现和利用，example\customerAnnotation\DemoTest.java
    中演示了一个日期校验的过程,详细代码说明请看：example\customerAnnotation\自定义注解说明.txt;
3.heading-thread：多线程学习；
4.heading-design：设计模式的学习；
4.1 代理设计模式,路径com\vi\design\proxy；
5.heading-netty:netty的demo
5.1 实现简单的netty应答；
5.2 实现netty http服务端和netty http客户端和非netty的http客户端请求应答；
